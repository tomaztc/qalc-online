Qalc quick reference

This browser calculator runs the real Qalculate! engine. Click any pink example to put it in the expression field. Enter evaluates; Shift+Enter inserts a newline; ↑ and ↓ recall earlier inputs. Settings and custom definitions are saved locally.

Official Qalculate! website libqalculate source qalc man page Full manual

Browser note: terminal history, screen clearing, exit, readline completion, signals, external commands, and plotting are unavailable. Use Clear for browser history. Exchange rates update through the web app; live preview is always handled separately and does not change qalc state.

// or divInteger division toward zero

~ | & << >> XORBitwise operators
= != < <= > >=Comparisons, equations, and inequalities
.; .* ./ .^Dot product and element-wise operations
∪ ∩ ∖ ⊖ ∈ ∉ ∋ ∌ ⊊ ⊆ ⊋ ⊇Set operations
:= or =Save a variable/function; := saves the unevaluated right side.

Priority is parentheses, powers of ten, exponentiation, functions, bitwise NOT, logical NOT, multiplication/division/remainder, parallel sum, addition/subtraction, shifts, comparisons, bitwise AND/XOR/OR, then logical AND/OR. Word operators need surrounding spaces.

Adaptive parsing (default) gives implicit multiplication priority unless spaces indicate otherwise: 1/5x is 1/(5x), while 1/5 x is (1/5)x. Conventional mode treats implicit and explicit multiplication alike; implicit-first always prioritizes adjacency; chain mode evaluates left to right; RPN uses postfix syntax.

Conversion, presentation, and replacement

Append to or -> to convert the whole expression. Prefix a unit with ? for an optimal decimal prefix, b? for an optimal binary prefix, + to force mixed units, or - to disable mixed units.

TargetAvailable forms
UnitsA unit/expression such as 20 miles / 2h to km/h, a variable/constant, base, optimal, prefix, or mixed.
Number basesbin/binary, bin#, oct, duo, hex, hex#, sexa/sexa2/sexa3, latitude/longitude variants, bijective, fp16/fp32/fp64/fp80/fp128, bcd, roman, time, unicode, base #, or bases.
Complex formrectangular/cartesian, exponential, polar, cis, angle/phasor.
Date/timeUTC, UTC+/-hh[:mm], or calendars.
Fractionsfraction, 1/n, decimals, or 1/#; prepend - for a simple fraction.
Notationsci/scientific, eng/engineering, or simple.
Algebra/outputfactors, partial fraction, or latex.

Append where or /. before any to clause for assignments, assumptions, and function replacement: x+y where x=1 and y=2, x^2=4 where x>0, or sin(5) where sin()=cos(). Strict leading assignments also work: x=1, y=2, x+y. Both operators apply to the entire expression regardless of parentheses.

Desktop qalc command line

These options describe the native CLI and do not apply to this browser page. Syntax: qalc [options] [expression]. Without an expression or file, qalc starts interactively.

-b, --base baseSet result and optional input base.
-c, --colorEnable expression/result colors and formatting.
--defaultsLoad default settings.
-e, --exratesUpdate exchange rates.
-f, --file fileExecute commands from a file first.
-h, --helpShow CLI options.
-i, --interactiveForce interactive mode.
-l, --list [term]List user-defined or matching objects.
--list-functions/--list-prefixes/--list-units/--list-variables [term]List one object category.
-m, --time millisecondsSet calculation/result-display timeout.
-n, --nodefsDo not load definitions from files.
--nocurrencies/--nodatasets/--nofunctions/--nounits/--novariablesSkip a global definition category.
-p [base]Programming mode: base conversion and caret-as-XOR.
-s, --set "option value"Apply a session setting.
-t, --tersePrint only the result.
-/+u8Disable/enable Unicode.
-v, --versionShow version.

Native interactive keybindings

Tab / Ctrl+IComplete functions, variables, and units, or insert the previous result when appropriate.
Ctrl+ASave the current result in a temporary variable.
Ctrl+DExit qalc.
Ctrl+ECycle exact, try-exact, and automatic approximation.
Ctrl+FCycle simple, decimal, and automatic fraction display.
Ctrl+LClear the terminal screen.
